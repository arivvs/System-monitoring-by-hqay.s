# System monitoring developed by hqay.s

Мониторинг системных ресурсов с графическим интерфейсом (GUI), написанный на Python.

## Основные возможности

* **Общая информация**: Визуальное отображение загрузки CPU, RAM и свободного места на диске с помощью прогресс-баров.
* **Фильтрация**: Программа автоматически скрывает процессы, потребляющие менее 0.1% ресурсов, чтобы не загромождать интерфейс.
* **Информация о GPU**: Отдельная вкладка для отслеживания состояния видеокарты (модель, загрузка, температура).
* **Обновление**: Обновление данных происходит раз в 2 секунды, что минимизирует нагрузку самой программы на систему.

## Технологии

* **Python 3.10+**
* **CustomTkinter** — для создания GUI.
* **psutil** — для сбора данных о системе и процессах.
* **GPUtil** — для мониторинга видеокарт NVIDIA.
